A game I'm sure you've heard of that is a lot more fun than it sounds is one where you go to Wikipedia and get from one Wikipedia page to the other only by clicking links on the page.
Here are a few examples you can use to challenge yourself.

  1. We'll start easy: try going from this page (paper) to this page (forest).
    Show Answer

    All you have to do for this is click on wood, then trees, and then there is a forest section in the tree page. This obviously isn't the only solution, just the one I found.

  2. Now let's do a couple harder examples to help you get the hang of things. First: go from the web design page to the thermostat page. (Using ctrl + f or cmd + f is allowed.)
    Show Answer

    Web Design -> User interface design -> Home appliances -> Air conditioning -> HVAC -> Thermostat

  3. Second: go from the darkness page to the United States Congress page.
    Show Answer

    Darkness -> Old english -> English language -> American english -> United States -> Congress

  4. Lets do a more specific destination. Try going from the United States Congress page where we ended off last time to the Kansas State University. This one is can be hard because Kansas State University is such a specific page, but it's pretty straightforward as long as you look a little harder.
    Show Answer

    United States Congress -> State -> Kansas -> Kansas State University
